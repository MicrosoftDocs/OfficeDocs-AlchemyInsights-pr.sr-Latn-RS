# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Dobiti nalog onemogućen greške u programu Outlook na Webu?

Ako dobijete grešku **Vaš račun je onemogućen** sa `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` kada pokušate da otvorite Outlook na Webu (poznatog i kao Outlook Web App), tvoj admin organizaciji možda je onemogućio vaš pristup programu Outlook na Webu. Tvoj admin mogu da biste otklonili ovu grešku tako što ćete pratiti korake u okviru teme, [Omogućavanje ili onemogućavanje Outlook Web aplikaciju za poštansko sanduče](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
