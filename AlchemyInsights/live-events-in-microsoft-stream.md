# <a name="live-events-in-microsoft-stream"></a>Događaji uživo u programu Microsoft stream

- [Kreiranje događaja uživo u programu Microsoft stream](https://docs.microsoft.com/stream/live-create-event)
- [Prvi koraci uz Microsoft timove uživo](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Konfigurisanje kodiranja za protok uživo u programu Microsoft stream](https://docs.microsoft.com/stream/live-encoder-setup)
