# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 pristup je odbijen, otkrivena je zloupotreba bezvredne pošte

Ova poruka se obično javlja kada korisnici šalju e-poruke sa IP adresa koristeći početni *. onmicrosoft.com* domen koji se dodeljuje novim zakupcima u programu Microsoft 365. Najlakši način da rešite ovaj problem jeste da:

1. [Dodajte domen zakupcu](https://docs.microsoft.com/microsoft-365/admin/setup/add-domain).

2. [Promena primarne e-adrese korisnika](https://docs.microsoft.com/microsoft-365/admin/add-users/change-a-user-name-and-email-address) u novi prilagođeni domen koji ste upravo dodali.
