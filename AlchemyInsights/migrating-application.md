# <a name="migrating-applications"></a>Migriranje aplikacija

[Resursi za Migriranje aplikacija u Azure Active Directory](https://docs.microsoft.com/azure/active-directory/manage-apps/migration-resources) – resursi koji će vam pomoći da migrirate pristup aplikacijama i potvrdu identiteta u Azure Active Directory (AZURE AD).

[Premeštanje potvrde identiteta aplikacije iz Active Directory usluga Federacije na Azure Active Directory](https://docs.microsoft.com/azure/active-directory/manage-apps/migrate-adfs-apps-to-azure) – projektovanje fokusirano usmeravanje za migraciju aplikacija iz Active Directory usluga Federacije na Azure Active Directory.
