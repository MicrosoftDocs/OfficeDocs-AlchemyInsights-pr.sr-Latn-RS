# <a name="install-project-professional"></a>Instaliranje programa Project Professional

Pogledajte članak [Instaliranje projekta](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) da biste saznali kako da instalirate verziju pretplate na Project Online i verziju koja nije pretplata na Project 2019, 2016 ili 2013. 

Za Project 2010 ili 2007, slijedite korake u [sistemu office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d)   ili [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Project se prodaje kao samostalnu aplikaciju, pa ako ga još uvek nemate, [kupite ili isprobajte Project](https://products.office.com/project).
