# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: rešavanje problema "obrada promena"

Ako vidite "obrada promena" u duže vreme, može da bude otvorena datoteka na mreži, broj datoteka zaglavljene u redu ili vrlo velika sinhronizacija, pogledajte [OneDrive: rešavanje problema "obrada promena"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) da bi se rešilo ovaj problem.

Takođe ćete videti ovu poruku ako imate datoteku koja se stalno menja (kao što je Outlook. PST datoteka ili baze podataka). Saznajte kako da [premestite Outlook datoteke sa podacima](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) koje redovno koristite iz usluge OneDrive.
