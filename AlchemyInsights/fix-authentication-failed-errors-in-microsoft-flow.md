# <a name="fix-authentication-failed-errors-in-microsoft-flow"></a>Otklanjanje grešaka "provjera autentičnosti nije uspelo" u Microsoft protoku

Ako dobijete poruku o grešci "potvrda identiteta nije uspela" kada pokušate da se prijavite u Microsoft protok, to je zato što je usluga protoka onemogućena u Azure aktivnom direktorijumu. Detalje i korake za popravljanje greške potražite [u članku ne možete da se prijavite u Microsoft protok i dobijate grešku "potvrda identiteta nije uspela"](https://support.microsoft.com/help/4316891).
