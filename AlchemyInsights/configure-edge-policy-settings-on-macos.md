# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>Korišćenje Jamf za Konfigurisanje postavki Microsoft Edge Policy na macOS

Da biste konfigurisali Microsoft Edge (stabilan kanal 81 ili noviju verziju) na Macosu, koristite datoteku manifesta Microsoft Edge smernica na lokaciji Jamf pro 10,19.

Više informacija potražite u članku [Konfigurisanje postavki Microsoft Edge Policy na macOS sa uslugom Jamf](https://go.microsoft.com/fwlink/?linkid=2134761).
