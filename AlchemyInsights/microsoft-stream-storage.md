# <a name="microsoft-stream-storage"></a>Skladište Microsoft toka

Stanari Microsoft protoka dobijaju fiksnu dodelu od 500 GB prostora za skladištenje i dodatnih 0,5 GB prostora za skladištenje po licenciranim korisnicima.
Više detalja o skladištu potražite u članku:

- [Microsoft pregled skladišta toka](https://docs.microsoft.com/stream/license-overview#storage)
- [Dodaj skladište u Microsoft tok](https://docs.microsoft.com/stream/storage-add-on)
- [Kvote i ograničenja u programu Microsoft stream](https://docs.microsoft.com/stream/quotas-and-limitations)
