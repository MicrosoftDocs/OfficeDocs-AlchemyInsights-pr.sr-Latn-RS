# <a name="site-is-not-available"></a>Lokacija nije dostupna

Ako vaša SharePoint ili OneDrive lokacija nije dostupna za više korisnika, moguće je da je privremena usluga problem. Proveri [kontrolne table zdravstvenih usluga](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) da vidim ako je uticala na tvoj stanar.
