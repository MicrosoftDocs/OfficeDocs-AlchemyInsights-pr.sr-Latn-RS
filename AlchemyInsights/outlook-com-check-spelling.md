# <a name="use-your-browser-to-check-spelling"></a>Koristite pregledač za proveru pravopisa

Iako je Outlook.com nema urok, pogledajte funkciju, mnogi web pregledači automatski u pravopisu, uključujući Microsoft Edge, Internet Explorer 10 i novije verzije, kao i trenutne verzije programa Firefox, Chrome i Safari. Proverite opcije za web pregledač da biste saznali više o tome kako da proverite pravopis.
