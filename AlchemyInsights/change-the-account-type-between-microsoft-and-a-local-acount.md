# <a name="change-the-account-type-that-you-sign-in-with"></a>Promena tipa naloga na koji se prijavljujete

Evo kako da promenite tip naloga koji koristite za prijavljivanje u Windows 10:

1. Izaberite " **Start** > "**Postavke** > **naloga** > za**informacije**.
2. Ako koristite lokalni nalog koji nije povezan sa e-adresom, ali želite da koristite Microsoft nalog, izaberite stavku **Prijavi se koristeći Microsoft nalog**.
3. Ako koristite Microsoft nalog koji je povezan sa e-adresom, ali želite da koristite lokalni nalog, umesto toga izaberite stavku " **Prijavi se koristeći lokalni nalog**".
4. Ako je potrebno, unesite lozinku, a zatim kliknite na dugme **dalje**.
5. Ponovo pokrenite uređaj. Kada se sledeći put prijavite, biće prikazan nalog koji ste zamenili.
