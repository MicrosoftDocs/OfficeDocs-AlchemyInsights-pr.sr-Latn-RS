# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 zabranjen pristup, stanar je premašila prag

Ova poruka se pojavljuje kada previše spam ili masovne pošte poslat je vaša organizacija Office 365. Da biste uklonili ovaj blok, treba da razume i objasniti uzrok agent za podršku, kao ispravili osnovni problem.
