# <a name="view-reports-on-sharepoint-activity-and-usage"></a>Prikaz izveštaja o aktivnosti sistema SharePoint i upotrebe

Idite na [stranicu izveštaji o korišćenju u Microsoft 365 centru administracije](https://admin.microsoft.com/AdminPortal/Home). U okviru **Izaberite izveštaj** odaberite izveštaj SharePoint aktivnosti ili korišćenje.

Više informacija potražite u članku [Osnovni podaci o SharePoint izveštajima](https://go.microsoft.com/fwlink/?linkid=875240).
