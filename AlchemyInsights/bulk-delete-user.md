# <a name="bulk-delete-user"></a>Korisnik masovnog brisanja

Veliki broj korisnika iz grupe možete ukloniti koristeći datoteku razdvojenu zarezima (CSV) u cilju masovnog brisanja korisnika. Da biste izbrisali masovne korisnike pomoću programa Azure AD portala, pogledajte temu " [masovno brisanje korisnika" u Azure aktivnom direktorijumu](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete).
