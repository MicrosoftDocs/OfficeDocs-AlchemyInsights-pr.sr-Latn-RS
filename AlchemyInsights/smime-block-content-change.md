# <a name="outlook-will-now-default-block-external-content-in-smime"></a>Outlook će sada podrazumevano blokirati spoljni sadržaj u S/MIME-u

Uvodimo dodatne bezbednosne mere u programu Outlook za Windows blokiranjem spoljnog sadržaja u S/MIME šifrovanim i potpisanim porukama po podrazumevanoj vrednosti i sprečavanju zamene po pojedinačnim porukama.

Za sve detalje pogledajte [Outlook za Windows blokira spoljni sadržaj u S/MIME porukama](https://support.office.com/article/2d3a4af1-fe41-475f-a888-fc7b997d112e).
