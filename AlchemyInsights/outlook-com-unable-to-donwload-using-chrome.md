# <a name="cant-download-files-in-chrome"></a>Nije moguće preuzeti datoteke u Chrome

**Izdaje** Svesni smo problema koji sprečava neke kupce da preuzmu datoteke kada koriste Google Chrome pregledač. 

**Rešenje** Koristite [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) ili neki drugi Web pregledač.
Saznajte više o [preuzimanju datoteka i Outlook.com priloga](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy).
