# <a name="removing-the-last-on-premises-exchange-server"></a>Uklanjanje prošle lokalne Exchange servera

Pre nego što uklonite poslednji Exchange serveru iz vaše lokalne organizacije, pobrini se razumete implikacije pomoći da odlučite ako ti treba taj poslednji server u neke druge svrhe. Za više informacija, pogledaj [kako i kada da a ako ne moћemo svoje lokalne Exchange servera u hibridni primene](https://technet.microsoft.com/en-us/library/dn931280%28v=exchg.150%29.aspx).
