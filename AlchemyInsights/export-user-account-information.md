# <a name="export-user-account-information"></a>Izvoz informacija o korisničkom nalogu

Da biste izvezli informacije o korisničkom nalogu u Microsoft 365 admin Center, uključujući informacije o licencama, smernicama za lozinke, blokiranim nalozima, statusu sinhronizacije AAD i još mnogo toga, idite na stranicu " **Korisnici** > [aktivnih korisnika](https://go.microsoft.com/fwlink/p/?linkid=834822) ", a zatim izaberite stavku " **Izvezi korisnike**".
