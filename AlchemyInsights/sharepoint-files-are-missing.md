# <a name="sharepoint-files-are-missing"></a>Nedostaju SharePoint datoteke

Ako se mnogo vaše OneDrive ili SharePoint datoteke brišu, prepišu, oštete ili zaražene malverom dajemo korisnicima mnoštvo opcija za pomoć u procesu oporavka.

Više informacija potražite u članku [Kako se oporavljaju stavke nestale, izbrisane ili oštećene u usluzi SharePoint Online i OneDrive for Business](https://go.microsoft.com/fwlink/?linkid=2110774).
