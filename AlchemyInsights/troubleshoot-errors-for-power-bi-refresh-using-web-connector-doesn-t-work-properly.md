# <a name="refresh-using-web-connector-doesnt-work-properly"></a>Osvežavanje pomoću Veb konektora ne radi ispravno

Ako imate scenarij Veb konektora koji koristi [Veb. stranicu](https://msdn.microsoft.com/library/mt260924.aspx) i ažurirali ste skup podataka ili izveštaj posle 18 novembra, 2016, morate da koristite mrežni prolaz da biste osvežite ispravno.

Za više informacija: [https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
