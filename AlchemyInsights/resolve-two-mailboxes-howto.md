# <a name="a-user-has-two-mailboxes"></a>Korisnik ima dva poštanskih sandučića

Hibridnog okruženja koje koriste Azure Active Directory povezivanje (AAD poveže) ili DirSync slučajno može prouzrokovati korisnik imati dva poštanskih sandučića: jedan u lokalnu organizaciju i jedan u oblaku.

Da biste rešili ovaj problem, vidim [Trajno jasan prethodni poštansko sanduče Info](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).
