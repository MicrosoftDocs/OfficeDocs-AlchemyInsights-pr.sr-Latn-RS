# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Podržana verzija programa SharePoint alatka za migraciju



Trenutno alatku za migraciju SharePoint podržava samo migracije iz SharePoint 2013. Verzija 2010 i 2016 nisu podržane u ovom trenutku.
  
Više informacija potražite u odeljku [uvođenje alatku za migraciju SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
