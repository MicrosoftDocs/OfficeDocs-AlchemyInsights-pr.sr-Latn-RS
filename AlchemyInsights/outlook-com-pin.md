# <a name="change-your-pin"></a>Promena PIN koda

Outlook.com ne koristi PIN, ali se možete upisati na računar sa PIN kodom ili Windows Zdravo. Da biste promenili PIN u operativnom sistemu Windows 10:

1. Kliknite na dugme Start ili pritisnite taster Windows na tastaturi, a zatim izaberite stavku **Postavke**  >  **Accounts**  >  **za prijavljivanje**naloga.
2. Kliknite na dugme **Promeni** u okviru **PIN koda**.
3. Unesite trenutni PIN, a zatim unesite i potvrdite svoj novi PIN kôd ispod.

[Pročitajte više o operativnom sistemu Windows Helo.](https://support.microsoft.com/help/17215/)
