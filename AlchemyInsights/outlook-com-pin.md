# <a name="change-your-pin"></a>Promena PIN-a

Outlook.com ne koristi PIN, ali se možete prijaviti na računar pomoću PIN koda ili Windows Hello. Da biste promenili PIN u operativnom sistemu Windows 10:

1. Izaberite dugme Start ili pritisnite taster Windows na tastaturi, a zatim izaberite **Postavke** > **Accounts** > **za prijavljivanje**naloga.
2. Kliknite na dugme " **Promeni** " u okviru stavke " **Zakači**".
3. Unesite trenutni PIN, a zatim unesite i potvrdite novi PIN ispod.

[Pročitajte više o usluzi Windows Hello.](https://support.microsoft.com/help/17215/)
