# <a name="deploy-office-apps"></a>Raspoređivanje Office aplikacije

Za primenu Office aplikacija više krajnjim korisnicima putem alatku za raspoređivanje Office, System Center Configuration Manager, kao deo sliku diska operativnog sistema, ili je u slučaju deljeni računar pomoću Windows Server sa udaljenom radnom površinom usluga (RDS) ili Terminal Usluge, razmislite o korišćenju vodič korak po korak u **Office 365 ProPlus savetnik za raspoređivanje**.
  
Da biste počeli, uverite se da ste prijavljeni kao admin stanar ili globalne admin, a zatim kliknite ovdje: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
