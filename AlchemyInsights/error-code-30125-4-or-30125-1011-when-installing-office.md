# <a name="error-code-30125-4-or-30125-1011-when-installing-office"></a>Kôd greške 30125-4 ili 30125-1011 prilikom instaliranja sistema Office

Možda ćete videti ovu grešku ako vaš antivirusni softver, zaštitni zid, proxy postavke ili veza sprečavaju instalaciju sistema Office. Evo nekih stvari koje možete probati da biste rešili problem:
  
- Ponovo pokrenite računar i ponovo instalirajte Office

- Instaliranje sistema Office pomoću [vanmrežnog instalacionog programa sistema Office](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA)

- Privremeno isključivanje postavki proxy servera

- Privremeno isključivanje antivirusnog programa

- Privremeno isključivanje zaštitnog zida

Za više informacija pogledajte [kôd greške 30125-4 ili 30125-1011 prilikom instaliranja sistema Office](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612?wt.mc_id=Alchemy_ClientDIA)
