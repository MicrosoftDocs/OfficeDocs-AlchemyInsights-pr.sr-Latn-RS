# <a name="47500-access-denied-please-try-again-later"></a>4.7.500 pristup je odbijen, pokušajte ponovo kasnije

Ova greška se javlja kada Količina saobraćaja e-pošte sa izvorne IP adrese premašuje ograničenje na osnovu reputacije (ili nedovoljne reputacije) izvorne IP adrese.

Blokiranje e-pošte sa izvorne IP adrese ističe za sat vremena. Ako je izvorna IP adresa lokalni e-server koji vam pripada, Verifikujte konfiguraciju konektora protoka pošte. Ako se ponašanje nastavi duže od sata, obratite se podršci da biste zahtevali izuzetak za izvornu IP adresu.
