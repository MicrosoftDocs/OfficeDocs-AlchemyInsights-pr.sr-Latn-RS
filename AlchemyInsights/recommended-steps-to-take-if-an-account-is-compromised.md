1. <span data-ttu-id="5af6e-104">[Poništite lozinku korisnika](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) odmah.</span><span class="sxs-lookup"><span data-stu-id="5af6e-104">[Reset the user's password](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) immediately.</span></span> <span data-ttu-id="5af6e-105">Ne šaljite krajnjem korisniku novu lozinku putem e-pošte.</span><span class="sxs-lookup"><span data-stu-id="5af6e-105">Do not communicate the new password through email to the end user.</span></span>

2. <span data-ttu-id="5af6e-106">Uklonite sve sumnjive [adrese za prosleđivanje](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) na nivou poštanskog sandučeta.</span><span class="sxs-lookup"><span data-stu-id="5af6e-106">Remove any suspicious [forwarding addresses](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) set at the mailbox level.</span></span>

3. <span data-ttu-id="5af6e-107">Uklonite sva sumnjiva [pravila prijemnog poštanskog sandučeta](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) postavljena u poštansko sanduče.</span><span class="sxs-lookup"><span data-stu-id="5af6e-107">Remove any suspicious [inbox rules](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) set within the mailbox.</span></span>

4. <span data-ttu-id="5af6e-108">Ako je korisniku onemogućeno slanje e-pošte, [idite na opciju „Korisnici sa ograničenim pristupom“ da deblokirate nalog](https://protection.office.com/?hash=/restrictedusers).</span><span class="sxs-lookup"><span data-stu-id="5af6e-108">If the user is blocked from sending email, [go to the Restricted Users to unblock the account](https://protection.office.com/?hash=/restrictedusers).</span></span> <span data-ttu-id="5af6e-109">Kada završite, korisniku bi slanje poruka trebalo da bude ponovo omogućeno u roku od jednog sata.</span><span class="sxs-lookup"><span data-stu-id="5af6e-109">Once done, the user should be able to resume sending messages within 1 hour.</span></span>

<span data-ttu-id="5af6e-111">Da biste umanjili potencijalno buduće curenje podataka ili ugrožavanje naloga, preporučujemo da čitate naš članak o [najboljim postupcima za Office 365 bezbednost](https://support.office.com/article/9295e396-e53d-49b9-ae9b-0b5828cdedc3).</span><span class="sxs-lookup"><span data-stu-id="5af6e-111">To minimize the potential of a data breach or a compromised account in the future, we recommend reading our [Office 365 Security best practices article](https://support.office.com/article/9295e396-e53d-49b9-ae9b-0b5828cdedc3).</span></span>
